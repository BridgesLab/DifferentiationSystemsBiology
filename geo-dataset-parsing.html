Dataset Acquisition

## Parsing....

This file was processed using R () and knitr (Andrew et al. 2013; Xie, 2013; Xie, 2013b).

The data was obtained from GEO using the dataset in GEO accession GSE6794, which is described in (Cheung et al. 2007). This file is described as:

3T3-L1 fibroblasts are a commonly used in vitro model for adipogenesis. When induced with hormones, they differentiate into mature fat cells. Here, microarrays were used to study 3T3-L1 adipose differentiation through time.

The file was downloaded, annotated and reformatted into an input file for GATE (MacArthur et al. 2009). This file was saved as Differentiation Time Course.clu. The file was downloades using bioMart (Durinck et al. 2009; Durinck et al. 2005). The annotation was done using GEOquery (Davis & Meltzer, 2007).
